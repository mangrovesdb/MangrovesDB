Blockchain Netwroks on MGDB
==============================

In this section we cover the important blockchain networks which are supported in the MangrovesDB are discussed. mangroves has the following essential features which makes it attaptable with almost all public and private blockchain network.

- It comes with **SQLite3 compatible interface** that can be integrated with any ``programming language`` and any ``3rd party dashboard/analytical services`` such as Grafana.
- ``dApp developers`` can easily write applications using ``SQL commands``.
- MGDB is **Self-deployable** and can connect to clients' ``private blockchain`` nodes.
- Mangroves has its own **CLI** to perform queries on real-time blockchain data.
- Mangroves is a **gateway** to ``all public/private blockchain networks``, such as Ethereum, Polygon, Hyperledger Hyperledger, etc.
- Develop Smart Contract leveraging relational database features.

The initial version of the mangroves can interact with Ethereum-compatible peojects, Solana and Hyperledger Fabric networks.
